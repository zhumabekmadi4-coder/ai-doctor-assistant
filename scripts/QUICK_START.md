# 🚀 Быстрый старт - Миграция на Supabase

## ⚡ Что нужно сделать ПРЯМО СЕЙЧАС

### Шаг 1: Выполнить SQL-скрипт в Supabase (5 минут)

1. Откройте ваш проект Supabase: https://supabase.com/dashboard
2. Перейдите в **SQL Editor** (левое меню)
3. Нажмите **New Query**
4. Откройте файл `update_database_schema.sql` в этой папке
5. Скопируйте **ВСЁ** содержимое файла
6. Вставьте в SQL Editor
7. Нажмите **RUN** (или Ctrl+Enter)
8. Дождитесь сообщения: "Структура базы данных успешно обновлена!"

### Шаг 2: Запустить миграцию шаблонов (1 минута)

Откройте терминал и выполните:

```bash
cd C:\Antigravity\ai-doctor-assistant
npx tsx scripts/migrate_local_templates.ts
```

Вы должны увидеть:
```
✅ Успешно мигрировано шаблонов: 3
```

### Шаг 3: Проверить результат (2 минуты)

1. Откройте Supabase Dashboard
2. Перейдите в **Table Editor**
3. Проверьте таблицы:
   - ✅ `templates` - должно быть 3 шаблона
   - ✅ `profiles` - должны быть колонки `credits`, `subscription_type`
   - ✅ `credit_transactions` - новая таблица
   - ✅ `subscription_plans` - должно быть 4 тарифных плана

### Шаг 4: Запустить локальный сервер (1 минута)

```bash
cd C:\Antigravity\ai-doctor-assistant
npm run dev
```

Откройте: http://localhost:3000

## 🔧 Если что-то пошло не так

### Проблема: SQL-скрипт выдает ошибку

**Решение**: 
1. Проверьте, что вы скопировали **весь** файл
2. Убедитесь, что у вас есть права администратора в Supabase
3. Попробуйте выполнить скрипт по частям (по одной секции)

### Проблема: Миграция шаблонов не работает

**Решение**:
1. Убедитесь, что SQL-скрипт выполнен успешно
2. Проверьте файл `.env.local` - должны быть правильные ключи Supabase
3. Проверьте, что таблица `templates` имеет все необходимые колонки

### Проблема: Не могу войти в систему

**Решение**:
1. Проверьте Supabase Authentication в Dashboard
2. Убедитесь, что пользователь создан
3. Проверьте настройки RLS (Row Level Security)

## 📊 Проверка системы кредитов

### Проверить баланс кредитов пользователя:

В Supabase SQL Editor выполните:

```sql
SELECT 
    email,
    full_name,
    credits,
    subscription_type,
    is_active
FROM profiles
WHERE email = 'ваш_email@example.com';
```

### Добавить кредиты вручную:

```sql
SELECT add_credits(
    (SELECT id FROM profiles WHERE email = 'ваш_email@example.com'),
    100,
    'Тестовое пополнение'
);
```

### Посмотреть историю транзакций:

```sql
SELECT 
    ct.*,
    p.email
FROM credit_transactions ct
JOIN profiles p ON ct.user_id = p.id
ORDER BY ct.created_at DESC
LIMIT 10;
```
